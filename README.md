# `@brillout/replace`

Replace `oldString` with `newString`:
```shell
replace oldString newString
```

Replace `old string` with `new string`:
```shell
replace "old string" "new string"
```

Replace `old "string"` with `new "string"`:
```shell
replace "old \"string\"" "new \"string\""
```

Show help:
```shell
replace --help
```
